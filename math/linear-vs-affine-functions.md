- For a single variable function:
	- **Linear Function**
		- A linear function is a type of affine function, but its b is always 0 as a linear function always goes through the origin 0.
			$$
			y(x) = ax + b 
			$$
		-  Hence it should always be described as 
		$$
		 y(x) = ax
		 $$
		- Here if x is zero the result is 0, preserving the origin.
		- To determine a linear function, only one point is needed $(x_{1}, y_{1})$ (because the other is always $(0, 0)$)
	- **Affine Function**
		- An affine function does not have its b as 0, and does not go through the origin 0.
		   $$
			f(x) = ax + b 
			$$
		- Here if the input is 0, the origin is still offset by b.
		 $$
           f(x)=b
           $$
	     - So the origin moves.
